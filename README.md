# Starbearer Portfolio

Local portfolio site for Teodora Jovanovic, built with React and Vite.

## Quick Start on Windows

For the easiest live preview:

1. Double-click [Start Portfolio Preview.bat](C:/Users/tilli/Downloads/starbearer-portfolio/Start%20Portfolio%20Preview.bat)
2. Wait for the browser to open `http://localhost:3000`
3. Leave that terminal window open while you review changes

For a production-style preview:

1. Double-click [Open Built Preview.bat](C:/Users/tilli/Downloads/starbearer-portfolio/Open%20Built%20Preview.bat)
2. Wait for the browser to open `http://localhost:4173`

## Terminal Commands

From [the project folder](C:/Users/tilli/Downloads/starbearer-portfolio):

```powershell
npm install
npm run dev
```

Useful extras:

```powershell
npm run dev:open
npm run check
npm run build
npm run preview:open
```

## How To Review Updates

Use `npm run dev` or `Start Portfolio Preview.bat` while changes are being made.
Vite hot-reloads automatically, so refreshing the browser is usually enough to see updates immediately.

Use `npm run build` plus `npm run preview` when you want to confirm the production version also looks right.

## If Node Was Just Installed

If `node -v` or `npm -v` still fail in Codex or your terminal:

1. Close and reopen the Codex app or terminal window
2. If it still fails, restart Windows once
3. Then run `node -v` and `npm -v` again

This is usually just a PATH refresh issue after installation.
